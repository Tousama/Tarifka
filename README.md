# Tarifka

This project is an application made with react-native that contains recipes.

<hr/>

## Requirements

- Visual Studio Code
- NodeJs
- Git
- Expo Go

## How can use?
- git clone https://github.com/Tousama/Tarifka.git
- cd Tarifka
- code .
- npm install
- npm start
- Use qr code with Expo Go or enter url manually
